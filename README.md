# CAP VIA – Avatar Oficial

Este repositorio registra el avatar oficial de **CAP VIA**, IA mentora de **JP Serra · VIA Sistemas Digitales**.

## Contenido

- `/img/master`: vistas maestras del avatar (frente, espalda, perfiles y 3/4).
- `/img/derivados`: recortes y versiones adaptadas a LinkedIn, GitHub, portadas, etc.
- `/docs/STYLEGUIDE.md`: lineamientos de uso (colores, proporciones, fondos permitidos).
- `/docs/CHANGELOG.md`: historial de cambios visuales.
- `/legal`: declaración de autoría y licencia.

## Propósito

- Dejar constancia pública y fechada de la creación del avatar CAP VIA.
- Centralizar el material maestro para todos los proyectos:
  - GEA – Gestión de Eventos Aduaneros
  - VIA – Función Integrada
  - ONE G9 / Back-End
  - Freelance & Marca Personal
  - Historias Digitales

Cualquier uso del avatar fuera de los proyectos de VIA requiere autorización expresa de **JP Serra · VIA Sistemas Digitales**.
